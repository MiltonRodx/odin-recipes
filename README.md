# Kitchen Recipes — Static Website

A simple static website showcasing kitchen recipes, built to practice and demonstrate fundamental HTML skills.

## Purpose

This project serves as a hands-on exercise for learning core HTML elements and document structure. By building a functional recipe site from scratch, I am practicing semantic markup, page organization, and standard HTML syntax.

## What This Demonstrates

Upon completion, this project will show proficiency in:

- Writing valid, semantic HTML
- Structuring content with standard elements (headings, paragraphs, lists, links, images)
- Organizing a multi-page static site
- Understanding the HTML document model and DOM basics

## Tech Stack

- **HTML5** — no frameworks, just plain markup

## Status

In progress. The site will feature multiple recipe pages with ingredients, instructions, and basic navigation.
